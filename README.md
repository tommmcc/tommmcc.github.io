# Personal Website – Tom McCarthy

This is a simple, minimal personal website with elements inspired by https://sive.rs/.

The goal is to create a clear, easy-to-use simple site that shares who I am, what I’m working on, and what I’m thinking about — without unnecessary clutter. It’s meant to feel personal, honest and useful.

---

## Design Approach

- Minimal layout  
- Mostly HTML and CSS  
- Little to no JavaScript  
- Mobile-friendly  
- No heavy frameworks  

If something doesn’t improve clarity, it doesn’t belong.

---
